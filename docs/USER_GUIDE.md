## 🚀 Швидкий старт

### 1. Встановлення

```bash
# Клонуємо репозиторій
git clone https://github.com/Kachalaba/Advanced-Swimming-Training-Bot-with-AI-features.git
cd Advanced-Swimming-Training-Bot-with-AI-features

# Створюємо віртуальне середовище
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# Встановлюємо залежності
pip install -r requirements.txt
```

### 2. Запуск веб-інтерфейсу

```bash
python3 -m streamlit run app.py
```

Відкрийте: **http://localhost:8501**

## 📹 Як аналізувати відео

### Крок 1: Завантажте відео
- Формати: MP4, MOV, AVI
- Рекомендована тривалість: до 60 секунд
- Якість: чим вища, тим краще

### Крок 2: Налаштуйте параметри

| Параметр | Опис | Рекомендація |
|----------|------|--------------|
| **FPS** | Кадрів на секунду | 10-15 (швидко), 30-60 (точно) |
| **Басейн** | Довжина в метрах | 25м або 50м |
| **Метод** | Тип аналізу | Гібридний (рекомендовано) |

### Крок 3: Запустіть аналіз
Натисніть **"Аналізувати відео"** і чекайте.

### Крок 4: Перегляньте результати
- 📊 Основні метрики (швидкість, темп)
- 🔬 Біомеханіка (кути суглобів, поза)
- ⏱️ Спліти (час на відрізках)
- 📹 Анотоване відео

## 🎯 Можливості системи

### Детекція та трекінг
- **YOLO** - знаходить плавця на кадрі
- **Velocity Prediction** - передбачає рух для точного трекінгу
- **Підводна детекція** - бачить плавця навіть під водою

### Біомеханіка (MediaPipe)
- 33 точки тіла
- Кути: лікті, коліна, плечі
- Вісь хребта та відхилення
- Оцінка гідродинаміки

### Аналіз швидкості
- Реальний timestamp з відео
- Точність спліту: ±0.001 сек
- Швидкість в м/с
- Темп в сек/100м

## ⚙️ Налаштування FPS

| FPS | Час обробки | Використання |
|-----|-------------|--------------|
| 5-10 | 1-2 хв | Швидкий перегляд |
| 10-15 | 2-3 хв | Тренування (рекомендовано) |
| 20-30 | 5-10 хв | Важливі відео |
| 30-60 | 10-20 хв | Фінальний аналіз |

## 🔧 Вирішення проблем

### Плавця не видно
- Збільшіть FPS до 30+
- Перевірте освітлення відео
- Система автоматично використовує підводну детекцію

### Спліти неточні
- Використовуйте FPS 15+
- Перевірте довжину басейну
- Спліти рахуються за реальним timestamp

### Помилка "Empty frame"
- Відео може бути пошкоджене
- Спробуйте інший формат (MP4)

## 📊 Розуміння результатів

### Posture Score (0-100)
- **80-100**: Відмінна техніка
- **60-80**: Добра, є що покращити
- **40-60**: Потребує роботи
- **0-40**: Серйозні проблеми

### Drag Coefficient
- **0.3-0.4**: Ідеальна обтікаємість
- **0.4-0.6**: Нормально
- **0.6+**: Високий опір води

### Streamline Score
- Чим ближче до 100%, тим краща горизонтальна позиція
