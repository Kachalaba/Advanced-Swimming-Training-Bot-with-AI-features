## 💡 Що це таке простими словами

Раніше, щоб розібрати техніку спортсмена, тренеру потрібні були дорогі датчики,
лабораторія або «око» й секундомір. **SPRINT AI робить це з одного відео.**

Ви знімаєте, як людина пливе, біжить, крутить педалі або виконує вправу —
а штучний інтелект **сам знаходить спортсмена, бачить кожен суглоб** і за секунди
видає зрозумілий звіт: що добре, що ні, і над чим працювати. Камера стає вашим
персональним біомеханіком.

> 🎯 Один інструмент для тренерів з тріатлону, плавання, легкої атлетики,
> фітнесу — **та фізичних терапевтів**.

## 🩺 Новинка: блок «Реабілітація»

Найцінніше оновлення — **режим відновлення руху (кінезіотерапія)** прямо у браузері.
Підходить для роботи з плечем, ліктем, стегном і коліном після травм.

**Що він уміє:**

| Можливість | Що це дає людині |
|---|---|
| 📹 **Live-камера** | Вмикаєте вебкамеру — і бачите аналіз **у реальному часі**, без завантаження файлів |
| 🦴 **Постуральна карта** | На екрані малюються осі **плечей, таза й корпусу** з підсвічуванням перекосів |
| 📐 **Амплітуда руху (ROM)** | Скільки градусів реально «проходить» суглоб — окремо ліворуч і праворуч |
| ⚖️ **Симетрія L/R** | Показує, наскільки одна сторона відстає від іншої — ключове у відновленні |
| 🎚️ **Калібрування кута камери** | Якщо телефон стоїть із нахилом — система це врахує й не зіпсує вимірювання |
| 🖥️ **Повноекранний режим** | Зручно для занять; вихід однією клавішею `Esc` |
| 💾 **Збереження в історію** | Кожне заняття лягає в картку атлета — видно прогрес тиждень за тижнем |
| ⚡ **Швидкість** | Потік оптимізований під сучасні ноутбуки (тестовано на MacBook Air M4) |

Також працює **аналіз завантаженого відео**: обираєте вправу → отримуєте відео
зі скелетом, графіки ROM і підсумок із симетрією.

> ⚕️ SPRINT AI — це тренувальний помічник, а не медичний прилад, і не замінює
> огляд лікаря чи фізичного терапевта.

## 🚀 Як запустити

Не лякайтеся команд — це робиться один раз. Потрібен лише встановлений
[Docker Desktop](https://www.docker.com/products/docker-desktop/).

**Сучасний веб-застосунок (рекомендовано):**

```bash
docker compose up --build
```

Відкрийте у браузері 👉 **http://localhost:3000**

**🖥️ Ярлик на робочому столі (macOS):** щоб запускати додаток подвійним кліком,
з іконкою на столі:

```bash
bash scripts/install-macos-launcher.sh
```

Деталі — у [`docs/macos-launcher.md`](docs/macos-launcher.md).

<details>
<summary><b>Альтернатива: класичний інтерфейс (Streamlit)</b></summary>

```bash
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python3 -m streamlit run app.py        # → http://localhost:8501
```
Тут доступні всі 8 вкладок із повним набором функцій.
</details>

## 🧠 Як це працює «під капотом»

1. **Знаходимо людину** на кожному кадрі (нейромережа YOLOv8) і «тримаємося» за
   потрібного спортсмена, навіть коли камера віддаляється.
2. **Будуємо скелет** із 33 точок (MediaPipe) і згладжуємо тремтіння.
3. **Рахуємо біомеханіку** — кути, амплітуди, симетрію, фази руху.
4. **Пояснюємо людською мовою** — звіт, графіки, поради AI-тренера.

Два інтерфейси спільно використовують один «движок» аналізу
(`video_analysis/`): сучасний **Next.js + FastAPI** застосунок і класичний
**Streamlit**-прототип.
