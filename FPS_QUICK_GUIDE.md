# ⚡ Быстрая шпаргалка по FPS

## 🎯 Какой FPS выбрать?

```
┌─────────────┬──────────────┬──────────────┬─────────────────┐
│ Сценарий    │ FPS          │ Время        │ Качество        │
├─────────────┼──────────────┼──────────────┼─────────────────┤
│ Тест        │ 5-10         │ ⚡ 1-2 мин   │ Базовое         │
│ Тренировка  │ 10-15 ✅     │ 🟡 2-3 мин   │ Хорошее         │
│ Соревнование│ 20-30        │ 🔴 5-10 мин  │ Отличное        │
│ Финал       │ 30-60 🔥     │ 🔴 10-20 мин │ Максимум        │
└─────────────┴──────────────┴──────────────┴─────────────────┘
```
